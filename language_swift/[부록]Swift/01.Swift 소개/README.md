# 01. Swift 소개 
- 애플 플랫폼 언어       
- 특징       
    **1. Modern**       
        - 최근에 생긴 좋은 기술만 업데이트한다.       
        - 깔끔하게 코딩가능      
        - 개발자 실수 줄이기위해 고도화 지속       
        - 자동 메모리 관리      
    **2. Safety**        
        - 안전하지 않은 클래스 제거      
        - 문법, 시스템을 통해 위험성 제거        
    **3. Fast**        
        - 더 빠른 속도로 기존 언어 대체       
 - 오픈소스 링크 ([링크](https://github.com/apple/swift))
